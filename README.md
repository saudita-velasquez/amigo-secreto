# Amigo Secreto

## Descripción

Este proyecto es una aplicación web simple que permite a los usuarios agregar amigos a una lista y sortear un "amigo secreto" de manera aleatoria. El objetivo principal es fortalecer las habilidades en lógica de programación y manipulación del DOM utilizando JavaScript.

## Funcionalidades

- **Agregar amigos**: Los usuarios pueden ingresar nombres de amigos en un campo de texto. Si el campo está vacío, se mostrará una alerta pidiendo un nombre válido.
- **Mostrar lista de amigos**: Cada vez que se agrega un amigo, la lista se actualiza automáticamente y muestra todos los amigos ingresados.
- **Sorteo de amigo secreto**: Una vez que hay al menos dos amigos en la lista, los usuarios pueden realizar un sorteo para seleccionar un amigo secreto de forma aleatoria.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Instrucciones de uso

1. Abre el archivo `index.html` en tu navegador.
2. Ingresa el nombre de un amigo en el campo de texto y presiona el botón para agregarlo a la lista.
3. Repite el paso 2 para agregar más amigos.
4. Una vez que tengas al menos dos amigos en la lista, presiona el botón para sortear un amigo secreto.
5. El nombre del amigo secreto seleccionado se mostrará en la pantalla.

## Ejemplo de uso

- Agrega amigos como "Juan", "María", "Pedro".
- Realiza el sorteo y obtendrás un resultado como "El amigo secreto sorteado es María".
